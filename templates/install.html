把網站變成 app,點一下就開,不用每次打網址

{# ===== iPhone 區塊 ===== #}

🍎 iPhone 使用者

第一步:用 Safari 打開
⚠️ 重點:必須用 Safari,不能用 LINE / FB 內建的瀏覽器

怎麼開 Safari?

  • 回到手機桌面
  • 找一個藍色指南針 icon 🧭(通常在底部那排)
  • 點下去就是 Safari
第二步:輸入網址

在 Safari 上方網址列打:swer-etf.zeabur.app

或點下面按鈕直接複製:

第三步:加到主畫面
① 點手機最下面正中間的 ⬆️(分享鈕)
② 跳出選單後往下滑,找「加入主畫面」
③ 右上角點「新增」
完成!回到桌面就有 app icon 了
{# ===== Android 區塊 ===== #}

🤖 Android 使用者

簡單版(推薦)
① 用 Chrome 打開 swer-etf.zeabur.app
② 進來第二次後,下面會跳「📱 一鍵變 app」按鈕
③ 點下去 → 系統跳安裝 → 確認
手動版(如果按鈕沒跳出來)
① 用 Chrome 打開網站
② 點右上角 ⋮(三個點)
③ 找「加到主畫面」或「安裝應用程式」
④ 點下去確認

常見問題

Q:我從 LINE 點連結進來能裝嗎?
A:不行。LINE 自己的瀏覽器不能裝 app(蘋果限制)。請先把網址複製,自己打開 Safari 或 Chrome 貼上。
Q:裝完桌面找不到圖示?
A:滑到桌面最後一頁找找,或從手機桌面下拉搜尋「ETF 觀察室」。
Q:要錢嗎?
A:完全免費。
