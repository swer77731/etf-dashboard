{# ===== iPhone 區塊 ===== #}

🍎 iPhone 使用者

最快的方法:用相機掃 QR Code
① 打開相機 app(不是 Safari,就是平常拍照那個相機)
② 對準上面的 QR Code
③ 點手機跳出來的網址
④ 自動用 Safari 開,跟著教學卡做就好

不用切瀏覽器、不用複製貼上!

進去 Safari 之後怎麼裝?
① 點右下角「⋯」三個點
② 選「分享」→「更多」
③ 找「加入主畫面」→「新增」
完成!回到桌面就有 app icon 了
{# ===== Android 區塊 ===== #}

🤖 Android 使用者

簡單版(推薦)
① 用 Chrome 打開 etf-watch.com
② 進來第二次後,下面會跳「📱 一鍵變 app」按鈕
③ 點下去 → 系統跳安裝 → 確認
手動版(如果按鈕沒跳出來)
① 用 Chrome 打開網站
② 點右上角 ⋮(三個點)
③ 找「加到主畫面」或「安裝應用程式」
④ 點下去確認

Q:我從 LINE 點連結進來能裝嗎?
A:不行。LINE 自己的瀏覽器不能裝 app(蘋果限制)。請先把網址複製,自己打開 Safari 或 Chrome 貼上。
Q:裝完桌面找不到圖示?
A:滑到桌面最後一頁找找,或從手機桌面下拉搜尋「ETF 觀察室」。
Q:要錢嗎?
A:完全免費。
