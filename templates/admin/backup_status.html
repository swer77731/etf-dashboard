備份制度說明

  • 備份檔位置: GitHub 私人 repo
  • 備份頻率: 每天 04:00、15:00(台北時間)各一次
  • 保留策略: 最近 90 天每天備份保留;每月 1 號額外永久保存(monthly/);每年 1/1 額外永久保存(yearly/)
  • 隱私保護: 備份檔內所有 email 欄位皆已 SHA-256 雜湊化,即使 GitHub 帳號被入侵也看不到原始信箱
  • 還原方式: 手動操作 — 從 GitHub 下載目標 .db.gz,解壓後上傳到 Zeabur Volume 覆蓋現有 etf.db
